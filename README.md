# PathTracer in RUST

* Parallelism through `scoped_threadpool`
* Access aligned bounding box and BVH acceleration structure
* Spheres and triangles
* Load OBJ files
* Next Event Estimation
* Russian Roulette
* Realistic materials
  * Absorbtion
  * Refraction
  * Reflection

* Only _one_ line of unsafe code (it reads floats from disk)
